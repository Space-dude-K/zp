Settings:

mainSettings:

saveDir - директория для экспорта готовых расчётников.

emailList - excel-список с ФИО, ID, email для привязки и отправки.

delayMinInSeconds - минимальный предел задержки в секундах перед повторной отправкой email.

delayMaxInSeconds - максимальный предел задержки в секундах перед повторной отправкой email.

exportFormat - формат для экспорта готовых расчётников. Поддерживается: .png.

exportRetryAttempts - количество попыток повторной обработки при неудачном разборе.

emailSettings:

smtpAddress - SMPT адрес почты.

smtpPort - SMPT порт почты.

email - email-отправитель.

emailPassword - пароль от почты (задаётся в приложении).

emailPasswordSalt - секция для расшифровки пароля (задается автоматически в приложении).

emailSubject - тема письма. Возможна вставка даты из расчётника (маска %parsedDate%).

emailText - содержание письма.

Interface:

Настройки - основные настройки, пароль для email задается через этот интерфейс.

Выход - выход из приложения.

Help - справка.

About - about.

Статус SMTP - наличие соединения с SMTP.

Список ID - наличие загруженного списка с ID.

Emails - количество корректных email / общее количество строк в списке ID.

Расчётников - количество обработанных расчётников с привязанным email / общее количество загруженных расчётников.

Отправлено - количество успешных отправленных email / общее количество email.

Загрузить список ID - загружается excel-список с ФИО, ID, email для привязки и отправки, автоматически загружается при старте, возможна повторная загрузка.

Выбрать файл - выбор excel-файла с оригиналом расчётника.

Загрузить расчётник - обработка расчётника. Возможна повторная обработка.

Отправить - отправка обработанных расчётников. Только при наличии соединения с SMTP и наличии корректных расчётников с привязанными ID-EMAIL.

Открыть архив - открытие saveDir в проводнике.

Show (в таблице обработанных расчётников) - открытие расчётнка в приложении по умолчанию, которое соответствует выбранному формату файла.
